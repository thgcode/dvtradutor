# Tradutor do sistema Dosvox
Este repositório visa abrigar o tradutor do sistema Dosvox, com adaptações para
compilar com o FPC (Free Pascal Compiler) e para funcionar com a pipeline que
estamos criando para o repositório de modificações do [Edivox](https://github.com/JosielSantos/edivox).

## Créditos
Este código foi produzido pela equipe do [Projeto Dosvox](http://intervox.nce.ufrj.br/dosvox/) (NCE/UFRJ). Fiz leves
modificações para compilar no FPC, mas o crédito todo deve ser dado a eles.
